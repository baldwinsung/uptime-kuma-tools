# Manage Uptime Kuma via API ( Uptime-Kuma-Web-API )

## Setup Virtual Env
```
mkvirtualenv uptimekuma
pip install uptime-kuma-api
```

## Monitor website via 443
```
./add_website.py
```

## Monitor hosts via ping
```
./add_hosts_ping.py
```
